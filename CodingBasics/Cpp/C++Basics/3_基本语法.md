# 基本语法组成

## `main()` 函数 —— 程序入口

```c++
#include <iostream> // 输入输出库

int main() {
    std::cout << "Hello, world!" << std::endl; // 输出语句
    return 0;
}
```

`#include <iostream>`：预处理指令，包含输入输出功能。

`std::cout`：输出到控制台（console output）。

`std::endl`：换行。

`return 0`：返回主函数执行成功。

## 条件语句

```c++
if (a > 0) {
    std::cout << "Positive" << std::endl;
} else {
    std::cout << "Non-positive" << std::endl;
}
```

## 循环结构

```c++
for (int i = 0; i < 5; ++i) {
    std::cout << i << " ";
}
```

## 函数定义

```c++
    return x + y;
}
//调用：

```

## 类与对象

```c++
class Person {
public:
    std::string name;
    int age;

    void introduce() {
        std::cout << "My name is " << name << ", age " << age << std::endl;
    }
};

int main() {
    Person p;
    p.name = "Alice";
    p.age = 22;
    p.introduce();
}
```
