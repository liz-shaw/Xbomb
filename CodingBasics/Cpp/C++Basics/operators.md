

# Arithmetic Operations 算术运算

- +

- -

- *

- /

  - 整数整除，浮点数浮点除

  - > [!NOTE]
    >
    > python:
    >
    > - / :浮点除法
    > - //:整除

- %

  - modulus

  - > [!NOTE]
    >
    > - 和python取整行为不同
    >
    >   - Python 的逻辑（Floor Division）
    >     - Python 倾向于让余数 $r$ 的符号与**除数 $n$** 保持一致。
    >     - **计算商：** $-3 / 2 = -1.5$。Python 使用 `floor()` 函数，将其向下取整为 **-2**。
    >     - **计算余数：** $-3 - (2 \times -2) = -3 + 4 =$ **1**。
    >
    >   - C++ 的逻辑（Truncated Division）
    >
    >     - C++（以及 Java、JavaScript 等）倾向于让余数 $r$ 的符号与**被除数 $a$** 保持一致。
    >
    >     - **计算商：** $-3 / 2 = -1.5$。C++ 直接截断小数部分，取整为 **-1**。
    >
    >     - **计算余数：** $-3 - (2 \times -1) = -3 + 2 =$ **-1**。

- > [!NOTE]
  >
  > python没有a++,++a这种东西

# Logical Operators 逻辑运算符

- AND: &&
- OR: ||
- NOT: !

> [!note]
> - 和python对比
> | **逻辑关系** | **Python** | **C++ / Java / JS** | **含义**                   |
> | ------------ | ---------- | ------------------- | -------------------------- |
> | **与 (AND)** | `and`      | `&&`                | 两者皆为真，结果才为真     |
> | **或 (OR)**  | `or`       | `||`                | 只要有一个为真，结果就为真 |
> | **非 (NOT)** | `not`      | `!`                 | 取反                       |
>
> - 位运算区分
>   - `5 && 2` $\rightarrow$ 结果是 **True** (逻辑判断)
>   - `5 & 2` $\rightarrow$ 结果是 **0** (位运算：`0101 & 0010 = 0000`)
